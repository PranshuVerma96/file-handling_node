# Node.js File Handling Project

This project demonstrates file handling in Node.js using the built-in **fs (File System)** module.

## 🚀 Features

* Create files
* Read file content
* Append data to file
* Copy files
* Delete files
* Get file information
* Create directories

## 📚 Concepts Used

* Node.js modules
* fs module
* Synchronous vs Asynchronous operations
* File system operations

## 🛠️ Technologies

* JavaScript (Node.js)

## ▶️ How to Run

```bash
node index.js
```

## 📌 Example Operations

* Writing data to file using `writeFileSync`
* Appending logs using `appendFileSync`
* Reading files using `readFileSync` and `readFile`
* Deleting files using `unlinkSync`

## 🎯 Purpose

This project is built for learning backend basics and understanding how Node.js interacts with the file system.
